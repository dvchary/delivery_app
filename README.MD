# To Check the node Version

```
node --version
```

# To initialize the Firebase Console

```
firebase init
```

# To Start the Server

```
npm run serve
```

# To Create the React Project using the npm & yarn

```
npm create-react-app folder_name
```

```
yarn create-react-app folder_name
```

# To Start the Client

```
yarn start
```
